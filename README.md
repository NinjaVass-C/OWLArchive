# OWLArchive
Introducing the OWL Archive, a personal project I am working on to preserve all match data for the former Esports Circuit Overwatch League. By the end of this project,
I hope to have all prior match data from all 6 seasons of the overwatch league neatly catalogued, allowing fans to see results from their favourite teams, easily access
VODs for prior matches, and relive the experiences myself and many others had before its retirement. The project is running on an Express.JS backend, connected to a MySQL
database with data scrapped from previous seasons. Vue.JS is in use for the front end as a single page application, ensuring all your Overwatch League search needs are in one 
place.
